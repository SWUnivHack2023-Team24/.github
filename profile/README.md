## BACKGROUND
노후 경유차, 언제든 자유롭게 탈 수 있어!
### INSIGHT
> 현재 소유중인 노후 경유차의 갑작스런 폐차 권고, 폐차를 하지 않고, 벌금도 안내면서 탈 수 있는 방법이 없을까?
- 오랫동안 운행한 차를 폐차 하는 아쉬움 + 폐차 권고로 인한 절망감 + 계속해서 발생하는 벌금

→ 노후 경유차 제도로 인한 불편함(운행 제한)을 해소하는 서비스를 기획하자!
